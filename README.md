# AI Cyber Security Detection System

This project is an AI-based cybersecurity system that detects malicious network traffic using machine learning.

## Features
- Real-time attack detection
- SOC-style dashboard
- Live traffic simulation
- Random Forest model
- CICIDS dataset

## Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- Chart.js

```bash
python main.py
